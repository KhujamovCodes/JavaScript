JavaScript

JavaScript (JS) dasturlash tilidir. Bu til orqali brauzerdagi web sahifalarga interaktivlik qo'shish mumkin. JS, HTML va CSS bilan birga ishlashda foydalaniladi. JS yordamida web sahifalarida ma'lumotlarni almashtirish, elementlarni yangilash, animatsiyalar yaratish, formalar bilan ishlashni tuzish, serverlar bilan ma'lumot almashish va kun tartibida turli amallar bajarish mumkin. Misol uchun, quyidagi bilan qanday bir marta "Salom Dunyo!" deb console.log() yordamida chiqarish mumkin. JavaScript keng tarqalgan va o'rgatilishi oson til hisoblanadi. Web-dasturchilar uchun juda muhim til hisoblanadi.

  • Frontend development - Javascript yordamida veb sahifaga dinamik ko’rinishga qo’shimchalar qo’shish, masalan matn va kontentni o’zgartish rasmlarni o’lchovini o’zgartirish va h.k. Undan tashqari Js fremvorklar yordamida veb sayt ishlab chiqish jarayonini ancha tez va oson amalga oshirish imkonyati mavjud.
  • Backend development -JS’ga asoslangan NodeJS texnologiya orqali yani Backend ya’ni sever-side dasturlashni amalga oshirish imkonyati mavjud.
  • Mobile application development - ReactNative va shunga o’xshash texnologiyalar orqali hybrid ko’rinishida mobil ilovalar yaratish imkonyati mavjud.
  • Game development - Online o’yinlar yaratishda Javascript ham faol tarzda ishlatiladi.

    O’zgaruvchilar (variable) - o’zida ma’lum bir qiymatni saqlovchi kontayner (container).

    let

    let -daturimiz ishlashi davomida mumkin bo’lgan o’zgaruvchilarni elon qilish uchun foydalinadi.

    Const

    const - dasturimz ishlashi davomida o’zgaruvchini o’zgarmaydigan o’zgaruvchilarni elon qiladi. const yordamida e’lon qilinadigan o’zgaruvchilar har doim boshlang’ich qiymatga ega bo’lishi shart.

    Javascript dasturlash tilida qiymatlar ikki guruhga bo’linadi: primitive va refrins.

    Primitive turdagi qiymatlar (Ular 8 xil bo’ladi)

    Primitive qiymat to’gridan to’g’ri o’zgaruvchi murojaat qiladigan joylashadi.

  • Number - sonlarni o’zida saqlovchi ma’lumot turi
  • String - o’zida belgilar ketma-ketligini saqlovchi malumot turi
  • Boolean - mantiqiy malumot turi bo’lib faqatgina true (rost) yoki false (yolg’on) qiymatini qabul qilishi mumkin bo’lgan malumot turi.
  • Undefined - hali aniqlanmagan malumot turi.
  • Null - bo’sh qiymatga ega bo’lgan ma’lumot turi.
  • Symbol - yagona (unique) va o’zgarmas qiymatga ega bo’luvchi malumot turi. Bir marta ishlatadigan narsa.
  • BigInt - katta sonlarni saqlashga mo’ljallandan ma’lumot turi

    • Javascript dynamik typing funksiyasiga ega bo’lib unig yordamida o’zgaruvchi saqlovchi ma’lumot turini ko’rsatmasdan ham qiymat saqlash mumkin. O’zgaruvchi saqlaydigan malumot turi avtomatik tarzda aniqlanadi.

    typeof operatoridan foydalanib o’zgaruvchi saqlayotgam malumot turini aniqlash imkonyati mavjud.

    Logical Operators Nimalar kiradi ?

  • | - or Agar biza turu bilan false or qiladigan bo’lsak tru qaytaradi or hardoim tru qidiradi.

  • ! -not "! -not" operatori tanlash uchun negatsiya (not) operatori sifatida foydalaniladi. Bu operator orqali ifoda qiymatini mos o'zgaruvchiga tenglashtirish mumkin.

    • Truthy value = 1, true, “ “,

    Falsy values = false, “”, Infinitiy, -Infinitiy, null, undefined, NaN.
